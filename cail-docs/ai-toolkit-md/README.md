## Key Features

- **Provocations-First:** Opens with critical questions to pose to students
- **Conceptual + Practical:** Balances theory (Derek Bruff framework) with practice (reusable activities)
- **Generous Attribution:** Cites scholars, links workshops, shares community resources
- **Progressive Layering:** Each section moves from high-level framing to actionable steps
