# Diction

*Word and vocabulary choices. Verb selection. Naming principles.*

---

## Vocabulary Bans (hard)

### AI slop — never use
delve, tapestry, vibrant, crucial, comprehensive, meticulous, embark, robust, seamless, groundbreaking, synergy, transformative, paramount, multifaceted, myriad, cornerstone, reimagine, empower, catalyst, invaluable, bustling, nestled, realm

### Business jargon verbs — replace immediately
utilize → use
leverage → use / apply / deploy
facilitate → let / allow / make possible
implement → build / set up / run
enhance → improve / sharpen / extend
optimize → tune / improve / fix
foster → build / support / grow
drive (business sense) → push / generate / cause

### Vague intensifiers — cut or specify
"with particular force" — specify why, or cut
"especially acutely" — cut
"with renewed urgency" — cut
"in an important way" — cut
If something applies particularly, name precisely why. If you can't, the intensifier is hiding a gap.

### Setup phrases — kill on sight
"What makes this interesting is..."
"Here's the thing:"
"The key insight is..."
"What's remarkable is..."
"It's worth noting that..."
Make the point. Skip the announcement.

## Naming Principles

**Name the mechanism, not the category.**
- "infrastructure" → the ranking algorithm, the data retention pipeline, the admin subpoena process
- "the system" → name what acts
- "the platform" → Reddit, Blackboard, Discord — name it

**Name specific instances.**
- "campus subreddits" → r/CUNY, r/Baruch
- "studies have shown" → name the study and author
- "users" → students, faculty, the people this actually involves

**Don't overfit the frame.**
- "legal risk" belongs to lawyers calculating exposure; CUNY students face "vulnerability" or being "findable"
- "extraction pressures" is analyst-speak; name the concrete behavior (data retention, behavioral logging)

**Quote platform terminology directly when it exists.**
- Reddit's own labels: "hottest," "controversial," "upvoted" — quote them; don't paraphrase
- The interface's own language grounds the claim in the actual object

## Verb Selection

Strong verbs for academic and analytical prose:
build, cut, break, run, ship, fix, test, push, pull, grab, drop, hit, show, reveal, compel, generate, push, embed, sort, mark, trace, assign, expose, produce, name, state, describe, demonstrate, account for

**Verb audit on every revision pass:**
For each sentence, ask: is the verb the most descriptively precise option? Weak copula constructions ("are X-mediated," "is shaped by") usually have a stronger replacement. Find the verb that carries the meaning rather than outsourcing it to an adjective.

Example: "participation trajectories are platform-mediated outcomes" → "participation trajectories encode the pressures of the infrastructure they run through."

**Passive voice:** Use selectively for institutional agents. "Users were compelled to hand over data" softens the company's role. "Reddit handed over data" puts the action on the actor. Choose based on what you want the reader to see.

## No Invented Academic Noun Phrases

LLMs routinely mint noun phrases that sound scholarly but appear in no actual literature: "LLM-derived predictability," "semantic retrieval interference," "distributional alignment pressure," "cloze-norming divergence."

Test: Would a researcher in the relevant field use this phrase in their own writing? If not, don't coin it. Use the terms that appear in the literature you're citing.

Fix: Describe the thing with a verb. "LLMs generate better regression predictors for reading-time data than offline cloze norms do" rather than "LLM-derived predictability outperforms cloze norming."
