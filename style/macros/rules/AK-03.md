# AK-03 — Citation hygiene

**Category:** Academic register  
**Added:** 2026-04-06  
**Non-negotiable:** No (required for academic profiles)

---

## Rule

No blog posts, aggregator sites, or secondary summaries doing load-bearing theoretical work. Peer-reviewed sources for key empirical and theoretical claims. Track edition, URL, and access date for online sources. When a claim depends on a source, cite that source — not a source that cites it.

---

## Source hierarchy

| Use for... | Source type |
|---|---|
| Empirical claims (study results, statistics, data) | Peer-reviewed journal or preprint with methods section |
| Theoretical frameworks | Peer-reviewed book or article by the framework's originator |
| Institutional facts (platform policies, legal documents) | Primary documents (transparency reports, court filings, platform ToS) |
| Historical context | Peer-reviewed secondary sources or primary documents |
| Supplementary illustration | Journalism, blogs — not as the sole source for any empirical claim |

---

## What "load-bearing" means

A source does load-bearing work when removing it would collapse the argument. A claim that requires the source to be true is load-bearing. A claim illustrated by the source (where the point already holds without it) is supplementary.

**Load-bearing:** "Reddit's compliance rate rose from 88% to 96% between 2012 and 2022" requires a source. The claim cannot stand without one.

**Supplementary:** "As one journalist noted, the policy change surprised long-time users" — the observation about user surprise is illustrative, not the argument's foundation.

---

## Citation form

Cite in the form the field uses. For DHQ: author-date inline with full reference in the bibliography. For source URLs: include the date accessed in the bibliography entry. For datasets: include the version and access method.

---

## Invented academic noun phrases

See AK-05. If a source used a term, cite it. If no source uses the term, don't use it.
