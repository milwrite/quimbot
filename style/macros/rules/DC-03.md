# DC-03 — Cut filler

**Category:** Diction  
**Added:** 2026-04-06  
**Non-negotiable:** No (strong revision habit)

---

## Rule

Cut phrases that add length without adding meaning. The sentence before the cut and after must say the same thing; if the cut changes meaning, it was never filler.

---

## Filler list

| Phrase | Cut to |
|---|---|
| in order to | to |
| due to the fact that | because |
| at this point in time | now |
| in the event that | if |
| it is important to note that | [delete; if important, the note is what's important] |
| it should be noted that | [delete] |
| in terms of | [restructure: "X in terms of Y" → "how X affects Y"] |
| with respect to | [restructure or cut] |
| for the purpose of | to |
| in light of the fact that | because / given that |
| as a matter of fact | [delete] |
| needless to say | [delete — if needless, don't say it] |
| the question of whether | whether |
| that which | what |

---

## Examples

**Before:**
> In order to understand the compliance record, it is important to note that the documents were obtained due to the fact that FOIA requests were filed.

**After:**
> To understand the compliance record, note that the documents came through FOIA requests.

---

## Principle

Every word earns its place. If a phrase can be removed and the sentence is the same or better, remove it.
