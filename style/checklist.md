# Revision Checklist

*The canonical checklist. Load for any revision pass.*  
*Source: consolidates `writing/style/checklist.md` + `writing/STYLE_GUIDE.md` + rule cards.*  
*Last updated: 2026-04-06*

---

## ⛔ Pre-Post Protocol (mandatory — no exceptions)

No revised prose posts to milwrite without completing this sequence.

**Step 1 — Draft**
Produce the revision.

**Step 2 — Mechanical audit (Pass 1)**
Run every category below on the draft. For each category: count violations found and fixed. No skimming — grep-level check.

**Step 3 — Fix all violations**
Address every item flagged in Pass 1.

**Step 4 — Re-audit (Pass 2)**
Run the checklist again on the fixed draft. Must come back clean before posting.

**Step 5 — Post with audit line**
Every submission includes a one-line audit summary immediately after the prose:
```
Checklist: [categories checked] | violations found: X | violations fixed: X | clean ✓
```

**Enforcement:** If either Quimbot or Petrarch posts revised prose without the audit line, the other calls it out. milwrite can call it out at any time.

## Category 1 — Voice and Register

- [ ] Does this sound like a person thinking, or a model generating?
- [ ] Is the register right for the venue (DHQ, Journalism Studies, essay)?
- [ ] Is the tone too smooth or too polished? (If yes: loosen one construction per paragraph)
- [ ] Are claims bounded to what the evidence shows?

## Category 2 — Structure

- [ ] Stakes clear by paragraph one?
- [ ] Does the opening start with the thing itself, or with background/context?
- [ ] Does the ending extend or reframe — not summarize?
- [ ] Is there one clear, statable claim the piece makes?
- [ ] Does each paragraph do one thing?

## Category 3 — Paragraph Level

- [ ] Evidence embedded in prose (not footnoted where it could be integrated)?
- [ ] At least one serious counterargument per major claim cluster?
- [ ] Transitions doing work, not announcing work?

## Category 4 — Sentence Level: Banned Constructions

Run mechanically. Grep if needed.

- [ ] No contrastive pivots ("not X but Y," "not just X," "more than X") → DC-02/DC-07
- [ ] No interrupted subjects ("Subject, [qualifier], verb") → SX-01
- [ ] No abstract subjects in short stub sentences → SX-04
- [ ] No bridging constructions ("is what," "is this:," "What X is Y") → SX-03
- [ ] No "What X does" openers → SX-05
- [ ] No "in this light" mid-sentence → (banned construction)
- [ ] No list subjects (two+ consecutive = hard stop) → SX-02
- [ ] No trailing -ing tag-ons → PR-04
- [ ] No anaphora repetition (same opener 3+ times) → PR-05
- [ ] No nominalizations as sentence openers → SX-04
- [ ] No bare "This" as subject → DC-05
- [ ] No bare demonstrative backward pointers ("That claim holds...") → DC-12

## Category 5 — Diction

- [ ] No AI slop vocabulary → DC-01
- [ ] No business jargon verbs (utilize, leverage, etc.) → DC-01/VB-02
- [ ] No vague intensifiers ("with particular force") → DC-10
- [ ] No invented academic noun phrases → AK-05
- [ ] No negative definitions → DC-11
- [ ] No filler ("in order to" → "to") → DC-03
- [ ] No setup phrases ("What makes this interesting is...") → DC-04
- [ ] No gap-bridging ("bridges the gap") → DC-08
- [ ] No melodramatic overclaims → DC-09
- [ ] No hanging quotations → PR-07

## Category 6 — Verbs

- [ ] Verb audit done? Each verb the most precise option? → VB-01
- [ ] No weak copula constructions that could carry more meaning → VB-01
- [ ] Logical connectors over conjunctive ones where applicable? → SX-06

## Category 7 — Punctuation

- [ ] Every em dash earned: closure test passed → PX-01
- [ ] Every colon introduces list or definition only → PX-02
- [ ] No punctuation stacking (consecutive heavy marks) → PX-03
- [ ] Semicolons audited: logical relationship surfaced → PX-04
- [ ] Single separator per clause → PX-05

## Category 8 — Citations and Claims

- [ ] No blog posts doing theoretical load-bearing work → AK-03
- [ ] Authors as agents (not findings): "Veldre et al. found X" → VB-03
- [ ] No false dilemmas in contrastive citation → CT-01/CT-02
- [ ] Data embedded at point of consequence, not as afterthought → AK-01
- [ ] Brackets [X] for unconfirmed data; no vague hedges ("many") → CL-02

## Category 9 — Prose Rhythm

- [ ] Sentence variation: no metronome cadence → PR-01
- [ ] Opener variation: no three same-structure openers in a row → PR-02
- [ ] Short sentence placed after longest sentence in each section → PR-03
- [ ] No freestanding echo sentences → PR-06
- [ ] One tricolon per argument cluster → PR-08

## Category 10 — Final Read

- [ ] Read one paragraph aloud. Does it breathe?
- [ ] One surprising, precise word per paragraph?
- [ ] Short sentence after the longest sentence in each section?
