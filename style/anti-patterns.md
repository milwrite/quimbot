# Anti-Patterns

*Consolidated list of what to cut on sight. Use as a scanning checklist.*

---

## Hard Kills (never appear in final copy)

| Pattern | Solution |
|---|---|
| Em dash | Period, comma, semicolon, or restructure |
| "Not X, but Y" / "not just X" | State Y directly |
| "What makes this interesting is..." | Make the point |
| "Here's the thing:" | Make the point |
| "The key insight is..." | Make the point |
| "delve, tapestry, vibrant, crucial, robust, seamless, groundbreaking, synergy, transformative, realm..." | Delete; find the real word |
| "in order to" | "to" |
| "due to the fact that" | "because" |
| Gap-bridging: "fills the gap between," "bridges the gap" | State what the paper does |
| "in this light" mid-sentence | The verb should carry it |
| "highlighting its importance" / trailing -ing significance phrases | Cut entirely |
| Invented academic noun phrases | Describe with a verb instead |
| "The finding that X shows Y" | "Author et al. found X" |
| False dilemma between studies with different domains | Present each study for what it actually shows |

## Sentence-Level Patterns

| Pattern | Before | After |
|---|---|---|
| Interrupted subject | "The second debate, equally consequential for what the procedure measures, concerned deletion method." | "The second debate concerned deletion method and carried equal weight for what the procedure actually measures." |
| Bridging construction | "Flocking is what happens when every bird runs..." | "Flocking happens when every bird runs..." |
| Nominalization | "pretraining's reduction of text to token distributions" | "pretraining reduces text to token distributions" |
| "What X does" opener | "What Gitelman and Jackson describe as X applies to..." | "Gitelman and Jackson describe X as it applies to..." |
| Abstract short sentence | "Meaning played no part in Harris's analysis." | "Harris set meaning aside entirely, arguing that..." |
| Negative definition | "patterns that no reader encounters as such" | "remediated as weighted co-occurrence patterns" |
| Logical preposition | "I trace these genealogies and interrogate the divergences..." | "I trace these genealogies to show where the divergences lie..." |
| Conjunctive where logical fits | "I kept each task separate, with no information carried forward." | "I kept each task separate so that no information carries forward." |
| Vague intensifier | "applies with particular force" | Name why, or cut the qualifier |

## Colon / Semicolon Anti-Patterns (from academic drafts)

| Before | After |
|---|---|
| "simple and automatable: delete words..." | "simple and automatable, requiring only that..." |
| "legible in the difficulty of restoration: when a player struggles..." | "legible in the difficulty of restoration, so that when a player struggles..." |
| "scales independently of blank count: early levels..." | "scales independently of blank count, with early levels..." |
| "But the purposes differ: Taylor..." | "The purposes diverge, since Taylor..." |

## Structural Anti-Patterns

| Pattern | Problem | Fix |
|---|---|---|
| Opening with definition | Reader doesn't care yet | Open with friction, concrete moment, or narrow question |
| Opening with survey ("scholars have long noted...") | Delays the actual move | Start with the specific claim or event |
| Anaphora overuse ("They assume that... They assume that...") | Mechanical, pattern-tic | Vary openers after the first instance |
| Tricolon pileup (3+ rule-of-three constructions in sequence) | Pattern-recognition failure | One tricolon per argument cluster |
| Summary ending | Collapses energy | Extend or reframe; don't restate |
| "Further research is needed" | Filler non-conclusion | Name the specific next question |

---

## Performance-Killing Academic Patterns

- Abstract analysis without personal or institutional stakes
- Process documentation without stakes or consequence
- Theory before practical application
- Opening with context or history instead of friction
- Claiming one study "contradicts" another when they address different domains
