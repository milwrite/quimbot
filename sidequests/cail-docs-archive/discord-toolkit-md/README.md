## Key Features

- **Progressive Disclosure:** Three-kit structure allows instructors to stop at any level
- **Procedural Granularity:** Step-by-step instructions with visual support (GIFs, screenshots)
- **Pedagogical Framing:** "Model Uses" paired with "Feature Suggestions"
- **Community Practice:** Attribution to contributors, external resources, shared templates
