## Slide 9 — Command Line Basics

**Label:** Tool
**Title:** Command Line Basics

**Stage (terminal):**
```
# where am I?
pwd

# list files
ls -la

# move into a folder
cd my-project

# go up one level
cd ..

# create a file
touch index.html

# make a new folder
mkdir assets
```

---

## Slide 10 — Versioning + Project Management with Git

**Label:** Tool
**Title:** Versioning + Project Management with Git

**Stage (terminal):**
```
# initialize a repo
git init

# stage your changes
git add .

# save a snapshot
git commit -m "first commit"

# push to GitHub
git push origin main

# view history
git log --oneline

# check what changed
git status
```
