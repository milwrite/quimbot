## Slide 7 — Versioning with Git

**Label:** Tool
**Title:** Versioning and Project Management with Git

**Stage (terminal):**
```
# initialize a repo
git init

# stage your changes
git add .

# save a snapshot
git commit -m "first commit"

# view history
git log --oneline

# check what changed
git status
git diff
```
