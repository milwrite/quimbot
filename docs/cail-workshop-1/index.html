Coding & Development Basics

Concept

Inference as Coding Scaffold

One-shot
Generate and run
Describe what you want. Receive code. Open in browser. That's the loop.
Deliberate
Decompose and iterate
Break the task into parts. Review each output. Use constraints to keep the model on track.
Concept

Problematics of Agentic Coding

The model can write all the code. That doesn't mean it should write all the code.
No-review iteration speeds things up. It also obscures what's happening — and why it breaks.

Command Line Basics

# where am I?
pwd

# list files
ls -la

# move into a folder
cd my-project

# go up one level
cd ..

# create a file
touch index.html

# make a new folder
mkdir assets
Tool

Versioning + Project Management with Git

# initialize a repo
git init

# stage your changes
git add .

# save a snapshot
git commit -m "first commit"

# push to GitHub
git push origin main

# view history
git log --oneline

# check what changed
git status
