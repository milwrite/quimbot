TS Factory Migration Demo

This page lives under the Quimbot GitHub Pages site as a quick stand-up wireframe for the creative-clawing TypeScript factory migration. It previews the current standalone artifacts that are being targeted for factory-style embedding.

Scope right now: keep standalone artifact pages public and linkable, while porting their render kernels into reusable JavaScript or TypeScript factory modules for embedded use elsewhere.
