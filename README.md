# Quimbot

Workspace for the Orchestra fine‑tuning project.

## Key Files
- `KANBAN.md` — task board (update after each action)
- `STATUS.md` — current state
- `DEVLOG.md` — timestamped work notes
- `RUNLOG.md` — run logs (log excerpts only)
- `COLLABORATION.md` — collaboration protocol
- `fine-tuning/` — scripts + workflows

## Data Policy
**Do not commit datasets** to Git. Large files belong in local `datasets/` or off‑repo storage. `.gitignore` excludes dataset directories.
